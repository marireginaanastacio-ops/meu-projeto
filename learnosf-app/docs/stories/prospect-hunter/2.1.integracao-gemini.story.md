## Story
**As a** usuário,
**I want** que o sistema gere automaticamente uma mensagem personalizada para cada lead usando IA,
**so that** economize tempo na criação de abordagens individuais e aumente a qualidade dos contatos.

## Acceptance Criteria
1. Endpoint `POST /api/leads/:id/generate-message` aciona a Gemini API com o perfil do lead
2. Prompt enviado ao Gemini inclui: nome, bio, plataforma do lead + descrição do serviço (de `data/config.json` ou fallback do `.env`)
3. Mensagem gerada é salva no campo `message` do lead em `data/leads.json` e `updated_at` é atualizado
4. Geração completa em menos de 5 segundos (NFR3)
5. Em caso de erro da API Gemini, retorna `{ success: false, error: "...", code: "GEMINI_ERROR" }` sem corromper `leads.json`
6. Retorna `{ success: true, data: { id, message } }` em caso de sucesso
7. Modelo utilizado: `gemini-1.5-flash` com `maxOutputTokens: 256`

## Dev Notes

### Gemini Config (Source: architecture/prospeccao-automatizada-architecture.md#5.3)
- Modelo: `gemini-1.5-flash` (rápido, barato, adequado para texto curto)
- Max output tokens: 256 (mensagens de até ~5 linhas)
- Timeout: 8s com retry único em falha transitória
- Inicialização:
```javascript
import { GoogleGenerativeAI } from '@google/generative-ai';
const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
const model = genAI.getGenerativeModel({
  model: 'gemini-1.5-flash',
  generationConfig: { maxOutputTokens: 256 }
});
```

### Template de Prompt Default (Source: architecture/prospeccao-automatizada-architecture.md#3)
Variáveis suportadas: `{{nome}}`, `{{bio}}`, `{{plataforma}}`, `{{service_description}}`

O template default está em `data/config.json`. Se vazio, usar `process.env.SERVICE_DESCRIPTION`.

### Endpoint (Source: architecture/prospeccao-automatizada-architecture.md#4)
```
POST /api/leads/:id/generate-message
Response: { "success": true, "data": { "id": "uuid", "message": "texto gerado" } }
```

### Importante: Não corromper JSON
Se a chamada ao Gemini falhar, o campo `message` do lead NÃO deve ser modificado.
Só salvar no JSON após obter resposta bem-sucedida.
