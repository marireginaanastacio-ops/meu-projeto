## Story
**As a** usuário,
**I want** ver todos os meus leads em uma lista com informações básicas,
**so that** tenha uma visão geral da minha base de prospects assim que abrir o dashboard.

## Acceptance Criteria
1. Projeto React + Vite + Tailwind CSS + shadcn/ui configurado e funcionando em `:5173`
2. Proxy Vite configurado: requisições para `/api` redirecionadas para `http://localhost:3001`
3. `services/api.js` implementado como wrapper de `fetch` para todos os endpoints do backend
4. Hook `useLeads()` implementado: busca leads via `GET /api/leads`, gerencia estado de loading/error
5. Página principal (`LeadsPage.jsx`) exibe lista de leads com cards contendo: nome, badge de plataforma colorido (rosa=Instagram, azul=LinkedIn), bio truncada (max 100 chars), badge de status colorido, data de captura formatada (pt-BR)
6. Estado vazio exibe mensagem "Nenhum lead encontrado" com instrução para realizar primeira busca
7. Interface completamente em português brasileiro
8. Build `npm run build` completa sem erros

### API Wrapper Pattern
```javascript
// client/src/services/api.js
const BASE_URL = '/api';

async function apiFetch(path, options = {}) {
  const res = await fetch(`${BASE_URL}${path}`, {
    headers: { 'Content-Type': 'application/json' },
    ...options
  });
  const data = await res.json();
  if (!data.success) throw new Error(data.error);
  return data;
}

export const getLeads = (params) => apiFetch(`/leads?${new URLSearchParams(params)}`);
export const getStats = () => apiFetch('/leads/stats');
// ... outros métodos
```
