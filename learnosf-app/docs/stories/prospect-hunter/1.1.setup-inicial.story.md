## Story
**As a** desenvolvedor,
**I want** inicializar o repositório com estrutura de pastas, dependências e variáveis de ambiente,
**so that** tenha uma base organizada e funcional para o desenvolvimento do Prospect Hunter.

## Acceptance Criteria
1. Repositório inicializado com `package.json` raiz com workspaces (`server/`, `client/`) e scripts de orquestração
2. Dependências instaladas em `server/`: `express`, `apify-client`, `@google/generative-ai`, `dotenv`, `uuid`, `cors`
3. Dependências instaladas em `client/`: `react`, `react-dom`, `vite`, `tailwindcss`, `@shadcn/ui`
4. Arquivo `.env.example` criado com variáveis: `APIFY_TOKEN`, `GEMINI_API_KEY`, `PORT`, `SERVICE_NAME`, `SERVICE_DESCRIPTION`
5. Estrutura de pastas criada conforme arquitetura: `server/src/{routes,controllers,services,middleware,utils}`, `client/src/{components,hooks,services,pages,utils}`, `data/`
6. Arquivo `data/leads.json` inicializado com `[]` e `data/config.json` com template padrão de prompt
7. `npm run dev` na raiz inicia servidor Express (:3001) e Vite (:5173) simultaneamente sem erros
8. Arquivo `.gitignore` inclui: `node_modules`, `.env`, `data/leads.json`, `data/config.json`

### Scripts Raiz (Source: architecture/prospeccao-automatizada-architecture.md#8)

```json
{
  "scripts": {
    "dev": "concurrently \"npm run dev:server\" \"npm run dev:client\"",
    "dev:server": "cd server && node --watch src/index.js",
    "dev:client": "cd client && vite",
    "build": "cd client && vite build",
    "start": "cd server && NODE_ENV=production node src/index.js",
    "test": "cd server && vitest run"
  }
}
```

### Vite Proxy (Source: architecture/prospeccao-automatizada-architecture.md#5.4)

```javascript
// client/vite.config.js
export default {
  server: {
    proxy: {
      '/api': 'http://localhost:3001'
    }
  }
}
```
